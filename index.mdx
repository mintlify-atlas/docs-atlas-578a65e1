## Overview

Progress Overlay is a Next.js application designed for streamers who want to display their GitHub project progress in real-time. Built with cutting-edge web technologies, it features stunning visual effects including ASCII art, curved marquee text, and animated overlays.

Perfect for developers streaming their coding sessions on Twitch, Progress Overlay automatically fetches and displays GitHub issues as a progress tracker, making it easy for viewers to follow along with your development journey.
