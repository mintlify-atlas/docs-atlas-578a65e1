## Component Library

The Progress Overlay project includes a collection of animated visual components built with React, Three.js, and SVG. These components are designed to create engaging Twitch overlays with interactive effects.

## Component Categories

### Visual Effects

- **ASCIIText** - Three.js-powered 3D ASCII text with shader effects
- **Noise** - Animated grain overlay for texture

### Progress Tracking

- **Progress** - Basic progress bar primitive
- **ProjectProgress** - GitHub issue tracking with progress visualization

### UI Components

- **CurvedLoop** - Curved text marquee with interactive dragging

## Usage

All components are client-side React components built with TypeScript. They're designed to be used in Next.js pages with the `"use client"` directive.

```tsx
import ASCIIText from "@/components/ASCIIText";
import CurvedLoop from "@/components/CurvedLoop";
import Noise from "@/components/Noise";

export default function MyOverlay() {
  return (
    <>
      <ASCIIText text="Hello World!" />
      <CurvedLoop marqueeText="Streaming Now" />
      <Noise />
    </>
  );
}
```
